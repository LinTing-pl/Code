蒙奇·D·路飞

蒙奇·D·路飞，日本漫画《航海王》及其衍生作品中的男主角。绰号“草帽”路飞，是草帽一伙、草帽大船团船长，四皇、极恶的世代之一。橡胶果实（人人果实·幻兽种·尼卡形态）能力者，悬赏金30亿贝里。梦想是找到传说中的大秘宝（ONEPIECE），成为海贼王。

罗罗诺亚·索隆

索隆是一名由伊露维塔在爱努的大乐章之前创造出来的迈雅。在时间初始的时候，他和其他爱努一起进入一亚。在那里他成为了被称作迈隆的奥力的迈雅。但不久之后他就被米尔寇诱入歧途，成为黑暗大敌强大也最被信任的仆从。

山治

山治，原名文斯莫克·山治，日本漫画《航海王》及其衍生作品中的角色。草帽一伙厨师，外号“黑足”，草帽一伙怪物三人组之一，与罗罗诺亚·索隆并称为蒙奇·D·路飞的左膀右臂，是一个足技高手。

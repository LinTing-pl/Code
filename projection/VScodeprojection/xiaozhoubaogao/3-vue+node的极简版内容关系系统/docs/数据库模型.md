### 电子书（books）

* id : int 
* title: varchar
* img : varchar  //封面图
* orderby : int   //排序权重



### 章（chapters）

* id : int
* title: varchar
* orderby : int
* book_id: int //外键，指向电子书



### 节（sections）

* id : int
* title: varchar
* orderby : int
* md_text : text //md格式富文本内容
* html_text : text //html格式文本内容
* chapters_id : int //外键，指向章



### 博客（blogs）

* id : int
* title : varchar
* img : varchar
* md_text : text //md格式富文本内容
* html_text : text //html格式文本内容



### 教学视频（videos）

* id : int
* title: varchar
* img : varchar
* iframe_url : varcahr  // B站视频地址



### 资源下载（resources）

* id : int
* title: varchar
* code ：varchar //提取码
* url : varchar //下载地址



### 用户（users）

* id : int
* username: varchar
* password: varchar
